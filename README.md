Deployment

- Vercel (recommended):
	1. Create a new project on https://vercel.com and import the GitHub repo above.
	2. Vercel will detect Create React App; use the default build command `npm run build`.
	3. Or use the CLI:
```bash
npm install -g vercel
cd files
vercel
```
